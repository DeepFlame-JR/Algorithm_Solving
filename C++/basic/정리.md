# 재귀 함수
- 자기 자신을 계속해서 불러오는 함수 (재참조하는 함수)
- 큰 문제를 부분 문제로 나눠서 풀 때 활용

#### 주의 사항
- 종료 조건 > 로직 > 다시 호출의 순서
    - 종료 조건이 없다면 무한히 호출됨
- 매개 변수가 계속 바껴야함. (사이클이 존재하면 안 된다)
- Overlapping Problem이 존재 > 시간이 많이 걸림

#### 예시

```cpp
int fact(int n){
    if(n == 1 | n == 0) return 0;
    return n * fact(n-1)
}

int fibo(int n){
    if(n == 1 | n == 0) return n;
    return fibo(n-1) + fibo(n-2);
}
```
